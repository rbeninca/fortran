Aula CSS

Propriedades Básicas

Exemplos práticos de propriedades comuns:

  • border: 1px solid #000;
  • width: 200px;, height: 100px;
  • margin: 10px;, padding: 20px;
  • color: #333;
  • background-color: #fafafa;
  • opacity: 0.8;
  • border-radius: 8px;
  • font-family: Arial;, font-size: 16px;, font-weight: bold;
  • text-transform: uppercase;

Seletores CSS

Básicos:

  • Elemento: p {}
  • Classe: .classe {}
  • ID: #id {}

Hierárquicos:

  • Descendente: ul li {}
  • Filho direto: div > p {}
  • Irmão adjacente: h2 + p {}
  • Irmão geral: h2 ~ p {}

Propriedades de Texto

  • font-family: 'Arial', sans-serif;
  • font-size: 14px;
  • font-weight: 700;
  • line-height: 1.5;
  • text-align: center;
  • text-transform: capitalize;
  • letter-spacing: 2px;

Cores e Fundos

  • color: #000;
  • background-color: #fff;
  • background-image: url('fundo.jpg');

Prática Guiada

Monte uma página integrando HTML e CSS usando as seguintes diretrizes:

  • Crie um menu horizontal com listas e estilize com CSS.
  • Monte áreas de conteúdo com div, usando propriedades de espaçamento e cores.
  • Aplique diferentes seletores CSS para estilizar títulos, parágrafos e botões.
