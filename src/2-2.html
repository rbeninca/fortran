CSS Box Model

O que é o Box Model?

O Box Model (modelo de caixa) do CSS é o modelo fundamental utilizado pelos navegadores para calcular o espaço que um elemento ocupa em uma página web.

Ele é composto por quatro partes:

  • Content (Conteúdo): O espaço onde o texto e as imagens aparecem.
  • Padding (Preenchimento): Espaço entre o conteúdo e a borda.
  • Border (Borda): Linha que envolve o padding e o conteúdo.
  • Margin (Margem): Espaço fora da borda, separando o elemento dos demais elementos.

Exemplo Prático

Observe a estilização desta caixa:

Conteúdo interno com padding, borda e margem definidos. 

.exemplo {
  border: 2px solid #000;
  padding: 10px;
  margin: 15px;
  background-color: #eaeaea;
}

Propriedades do Box Model

  • width e height: definem largura e altura do conteúdo.
  • padding: adiciona espaço interno ao redor do conteúdo.
  • border: adiciona borda em torno do padding.
  • margin: adiciona espaço externo ao redor do elemento.
  • box-sizing: altera o cálculo do tamanho da caixa (content-box ou border-box).

Exemplo de uso do box-sizing para facilitar a medição dos elementos:


.elemento {
  box-sizing: border-box; /* Inclui padding e borda no cálculo da largura/altura */
  width: 200px;
  padding: 10px;
  border: 5px solid #000;
}
