Aula 1 - Introdução ao HTML

O que é HTML?

Uma página da web é um conjunto de arquivos que o navegador renderiza para exibir conteúdo. O HTML (Linguagem de Marcação de Hipertexto) é utilizado para estruturar esse conteúdo de forma semântica. Ele define títulos, parágrafos, imagens, links e muito mais.

HTML não é uma linguagem de programação, mas sim uma linguagem de marcação: ela marca o conteúdo com significado, como um título ou um parágrafo, permitindo que o navegador o exiba de forma adequada.

Estrutura de um Elemento HTML

Os elementos HTML são formados por uma tag de abertura, o conteúdo, e uma tag de fechamento. Exemplo:

<h1>Título</h1>
alt="Explicação de tags HTML">

Exemplos de Elementos HTML

  • <h1>: título principal
  • <p>: parágrafo
  • <a>: link
  • <img>: imagem
  • <button>: botão
  • <em>, <strong>: ênfases
  • <ul>, <ol>, <li>: listas
  • <table>: tabela
  • <form>: formulário
  • <div>, <span>: containers

Comentários

Comentários são trechos ignorados pelo navegador:

<!-- Este é um comentário -->

Atributos

Atributos fornecem informações extras às tags. Exemplo:

<a href="https://ifsc.edu.br">Site do IFSC</a>
<img src="foto.jpg" alt="descrição da imagem">

Aninhamento de Elementos

Tags podem ser aninhadas. Exemplo:

<h1>Título com <em>ênfase</em></h1>

Bloco vs Inline

Elementos de bloco ocupam toda a largura e começam em nova linha. Inline ficam na mesma linha.

Estrutura Básica de uma Página HTML

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
  </head>
  <body>
    Conteúdo aqui!
  </body>
</html>

Git e Controle de Versão

O Git é um sistema de controle de versão distribuído que permite salvar o histórico de mudanças no código.

  • commit: salva o estado atual do projeto com uma mensagem descritiva.
  • branch: uma linha de desenvolvimento paralela ao código principal.

Serviços como GitHub, GitLab e Bitbucket permitem hospedar seus repositórios e colaborar com outras pessoas.

GitHub Pages é um recurso do GitHub que permite publicar páginas web diretamente de um repositório.

CSS Básico

CSS é a linguagem usada para definir estilos em documentos HTML.


p {
  color: blue;
  font-size: 16px;
}

CSS Prático

Você pode aplicar estilos diretamente aos elementos com várias propriedades úteis:

  • border: define bordas
  • width, height: largura e altura
  • margin, padding: espaçamento externo e interno
  • color: cor do texto
  • background-color: cor de fundo
  • opacity: transparência
  • border-radius: cantos arredondados
  • font-family, font-size: fonte e tamanho
  • text-transform: maiúsculas, minúsculas etc.

Prática Guiada

Durante a prática, você deve montar uma página utilizando os seguintes recursos:

  • Listas: <ul>, <ol>, <li>
  • Tabelas: <table>, <tr>, <td>
  • Formulários: <form>, <input>, <textarea>, <button>, <label>, <select>, <option>
  • Estilos aplicados com CSS: border, padding, margin, color, etc.

Experimente integrar os conhecimentos em um site pessoal simples.
